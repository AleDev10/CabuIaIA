# 🤖 CabuIaIA

Uma extensão para o navegador Chrome que utiliza recursos de IA para interpretar o contexto de uma página web.

## 📋 Sobre o Projeto

CabuIaIA é uma extensão inteligente que integra capacidades de Inteligência Artificial diretamente no seu navegador Chrome, permitindo análise e interpretação automática do conteúdo das páginas que você visita.

## ✨ Características

- 🧠 Interpretação inteligente de conteúdo web usando IA
- 🌐 Integração nativa com Chrome
- ⚡ Interface leve e rápida
- 🔒 Processamento de contexto de páginas

## 🛠️ Tecnologias Utilizadas

- **JavaScript** (54.9%) - Lógica principal da extensão
- **CSS** (33.5%) - Estilização da interface
- **HTML** (11.6%) - Estrutura das páginas

## 📦 Dependências

- `@types/chrome: ^0.2.2` - Tipos TypeScript para Chrome API

## 🚀 Como Instalar

1. Clone o repositório:
```bash
git clone https://github.com/AleDev10/CabuIaIA.git
cd CabuIaIA
```

2. Instale as dependências (opcional):
```bash
npm install
```

3. Carregue a extensão no Chrome:
   - Abra `chrome://extensions/`
   - Ative o "Modo de desenvolvedor" (canto superior direito)
   - Clique em "Carregar extensão sem empacotamento"
   - Selecione a pasta do projeto

## 📝 Como Usar

1. Acesse qualquer página web
2. Clique no ícone da extensão CabuIaIA na barra de ferramentas
3. A extensão analisará o conteúdo da página usando IA
4. Visualize os resultados da interpretação

A extensão é configurada através do arquivo `manifest.json`, que define:
- Permissões necessárias
